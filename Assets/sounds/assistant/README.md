# Assistant Voice-Line Assets

Optional Waifu/anime assistant voice-line WAV or MP3 files can be placed here.
WAV is preferred if both formats exist for the same line.

- `waifu_ready.wav`
- `waifu_working.wav`
- `waifu_success.wav`
- `waifu_error.wav`
- `waifu_reject.wav`
- `waifu_deals.wav`
- `waifu_category.wav`

The same names are also supported with `.mp3`. For error/failure lines,
`waifu_failure.wav` and `waifu_failure.mp3` are accepted aliases.

If a file is missing, the extension skips that optional voice line and keeps the existing assistant text and standard sound effects.
